# CAN(Controller Area Network)

## 概要

自動車などで使われる通信方式。ノイズに強く、マスターレスで通信できます。

一般的なモジュールはSPI接続になります。

M5StickCだと通常は使わないプロトコルです。関数も準備されていますが、普通は使わない内部関数のみで、利用例もありません。
