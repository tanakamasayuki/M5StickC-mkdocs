# モーター制御(MCPWM)

## 概要

パルスを利用してモーターを制御する機能です。2種類のMCPWMユニットがあり、各ユニットには3つのPWM出力があります。

M5StickCだと通常は使わない機能です。関数も準備されていますが、普通は使わない内部関数のみで、利用例もありません。
