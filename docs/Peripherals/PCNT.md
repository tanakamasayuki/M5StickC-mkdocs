# パルスカウンタ(PCNT)

## 概要

入力信号の立ち上がり、または立ち下がりエッジの数をカウントできる2つのチャンネルを持つ機能です。

M5StickCだと通常は使わない機能です。関数も準備されていますが、普通は使わない内部関数のみで、利用例もありません。
