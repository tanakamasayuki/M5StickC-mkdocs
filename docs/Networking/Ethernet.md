# Ethernet(有線LAN)

M5StickCはEthernetを内蔵していないので、別途接続する必要があります。

SPI接続のEthernetモジュールや、UART接続のEthernetモジュールがありますが、接続性を考えるとEthernetポートのあるWiFiルーターを準備したほうが楽なケースが多いはずです。
