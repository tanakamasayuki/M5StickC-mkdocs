# HTTP Server

M5StickCでHTTPサーバーを動かすことができます。

ブラウザから設定を行う場合に使うことが多いですが、スマートスピーカーなどと連携してM5StickCの赤外線リモコンを操作したり、接続しているデバイスを操作することも可能です。
